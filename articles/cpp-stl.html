STL 容器与算法实战

C/C++ 2026-02-10 · 14 分钟阅读

掌握 vector、map、set 等核心容器，理解迭代器模式与算法组合的最佳实践。

序列容器

std::vector

动态数组，最常用的容器。支持随机访问，尾部插入效率高。

std::vector<int> v = {1, 2, 3};
v.push_back(4);           // 尾部添加
v.pop_back();             // 尾部删除
int x = v[0];             // 随机访问 O(1)
int y = v.at(0);          // 带边界检查

std::deque

双端队列，头尾操作都是 O(1)。

std::deque<int> d;
d.push_front(1);  // 头部添加
d.push_back(2);   // 尾部添加

std::list

双向链表，任意位置插入删除 O(1)，但不支持随机访问。

关联容器

std::map / std::unordered_map

std::map<std::string, int> m;
m["apple"] = 1;
m.insert({"banana", 2});

// 查找
auto it = m.find("apple");
if (it != m.end()) {
    std::cout << it->second;
}

// 遍历
for (const auto& [key, value] : m) {
    std::cout << key << ": " << value;
}

std::set / std::unordered_set

存储唯一元素的集合，自动排序。

std::set<int> s = {3, 1, 2};
s.insert(4);          // 插入
s.erase(1);           // 删除
bool has = s.count(2); // 查找

常用算法

#include <algorithm>

std::vector<int> v = {3, 1, 4, 1, 5};

// 排序
std::sort(v.begin(), v.end());

// 查找
auto it = std::find(v.begin(), v.end(), 4);

// 二分查找（需先排序）
bool found = std::binary_search(v.begin(), v.end(), 3);

// 计数
int cnt = std::count(v.begin(), v.end(), 1);

// 去重（需先排序）
auto last = std::unique(v.begin(), v.end());
v.erase(last, v.end());

最佳实践

  • 默认使用 vector，除非有特殊需求
  • 频繁查找用 unordered_map，需要有序用 map
  • 使用范围 for 循环代替迭代器遍历
  • 使用 emplace_back 替代 push_back
  • 传递容器时使用 const 引用
