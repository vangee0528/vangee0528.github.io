## 本地预览

```bash
# 克隆仓库
git clone https://github.com/vangee0528/vangee0528.github.io.git

# 进入目录
cd vangee0528.github.io

# 启动本地服务器（需要 Python）
python -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后访问 http://localhost:8000

## 部署

本博客使用 GitHub Pages 部署：

1. 创建仓库 `vangee0528.github.io`
2. 推送代码到 main 分支
3. 在仓库设置中启用 GitHub Pages
4. 访问 https://vangee0528.github.io
