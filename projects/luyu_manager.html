炉羽饰品生产管理系统

基于 Flutter 开发的跨平台生产管理应用，专为饰品生产企业量身定制。 系统采用完全本地化设计，支持离线运行，提供员工管理、产品管理、生产记录、 工资统计等核心功能，助力企业实现数字化管理转型。

炉羽饰品生产管理系统是一款面向小型饰品生产企业的管理工具。针对企业实际业务需求， 系统提供了从员工信息管理到工资结算的完整解决方案，采用现代化技术栈构建， 支持 Windows 桌面端和 Android 移动端双平台运行。

系统设计遵循「离线优先」原则，所有数据存储在本地 SQLite 数据库中， 无需网络连接即可正常使用所有功能。同时提供 WebDAV 云备份支持， 确保数据安全性与可恢复性。

核心功能模块

👥
员工管理

灵活的员工信息管理，支持两种工资计算模式，满足不同岗位需求。

  • 计件工资模式：按产量计算，支持工资系数调整
  • 计时工资模式：按时长计算，自定义时薪标准
  • 员工信息增删改查，支持批量操作
  • 软删除机制，数据可从回收站恢复
  • 员工生产记录关联查询
📦
产品管理

产品分类与定价管理，支持多种产品类型和自定义单位。

  • 产品分类：单颗、链条、定位彩等
  • 自定义计量单位（个、条、套等）
  • 基准单价设置，支持随时调整
  • 产品描述与备注信息
  • 产品生产统计与分析
📝
生产记录

灵活的生产记录管理，支持员工单日多产品生产场景。

  • 一条记录可包含多个生产明细项
  • 自动关联产品单价，支持手动调整
  • 按日期、员工筛选查询
  • 生产记录详情查看与编辑
  • Excel 数据导入支持
💰
工资统计

自动计算员工工资，支持多种维度的统计分析。

  • 按时间范围统计（日/周/月/自定义）
  • 按员工、产品分类汇总
  • 计件工资 = 数量 × 单价 × 系数
  • 计时工资 = 时长 × 时薪
  • 一键导出 Excel 报表
💾
数据备份

双重备份机制，确保数据安全无忧。

  • 本地备份：自动生成 ZIP 压缩包
  • WebDAV 云备份：支持坚果云等服务
  • 智能备份管理：自动清理过期备份
  • 一键恢复：从备份文件快速还原
  • 备份设置：定时自动备份
⚙️
系统设置

丰富的系统配置选项，满足个性化需求。

  • 主题切换：浅色/深色模式
  • WebDAV 连接配置与测试

业务流程

以下展示系统核心业务场景的操作流程：

计件员工生产记录流程

1

创建员工信息

录入员工姓名，选择「计件」类型，设置计件工资系数（默认为 1.0）

2

录入产品信息

添加产品分类、名称、计量单位和基准单价

3

记录生产数据

选择员工和生产日期，添加当日生产的各产品数量，系统自动计算金额

4

查看工资统计

在报表页面选择时间范围，查看各员工工资汇总，导出 Excel 报表

数据备份流程

1

配置 WebDAV

在设置页面填写 WebDAV 服务器地址、用户名、密码，测试连接

2

创建备份

点击「立即备份」，系统将数据库打包为 ZIP 文件并上传到云端

3

恢复数据

从本地或云端选择备份文件，一键恢复数据库

数据库设计

系统采用 SQLite 关系型数据库，包含 4 张核心数据表：

员工表 employees

id
TEXT PRIMARY KEY 员工唯一标识，格式：YG_UUID
name
TEXT NOT NULL 员工姓名
type
TEXT NOT NULL 员工类型：'计件' 或 '计时'
pieceRateFactor
REAL DEFAULT 1.0 计件工资系数，用于调整最终工资
hourlyRate
REAL DEFAULT 0.0 时薪标准（元/小时）
deletedAt
TEXT 软删除时间戳，NULL 表示未删除
createdAt
TEXT NOT NULL 创建时间

产品表 products

id
TEXT PRIMARY KEY 产品唯一标识，格式：CP_UUID
category
TEXT NOT NULL 产品分类（单颗、链条、定位彩等）
name
TEXT NOT NULL 产品名称
unit
TEXT DEFAULT '个' 计量单位（个、条、套等）
unitPrice
REAL NOT NULL 基准单价（元）
description
TEXT 产品描述

生产记录表 production_records

id
TEXT PRIMARY KEY 记录唯一标识，格式：PR_UUID
employeeId
TEXT 关联员工 ID (外键)
recordedAt
TEXT NOT NULL 生产日期
notes
TEXT 备注信息

生产明细表 production_items

id
TEXT PRIMARY KEY 明细唯一标识，格式：PI_UUID
recordId
TEXT NOT NULL 关联生产记录 ID (外键)
productId
TEXT 关联产品 ID (外键，可选)
settledPrice
REAL NOT NULL 结算单价（记录时的单价快照）
quantity
REAL 生产数量（计件用）
hoursWorked
REAL 工作时长（计时用，单位：小时）

技术挑战与解决方案

跨平台 SQLite 支持

Flutter 默认的 sqflite 插件主要面向移动端，桌面端需要额外配置。 项目采用 sqflite_common_ffi 实现 Windows/macOS/Linux 平台的 SQLite 支持， 通过运行时检测平台自动切换实现：

if (Platform.isWindows || Platform.isLinux || Platform.isMacOS) { sqfliteFfiInit(); databaseFactory = databaseFactoryFfi; }

数据库版本迁移

随着功能迭代，数据库结构需要升级。项目实现了完善的版本迁移机制， 支持从旧版本平滑升级，确保用户数据不丢失：

Future<void> _upgradeDatabase(Database db, int oldVersion, int newVersion) async { if (oldVersion < 2) { // 添加新字段 await db.execute('ALTER TABLE products ADD COLUMN unit TEXT'); } if (oldVersion < 3) { // 修改约束需要重建表 await db.execute('CREATE TABLE production_items_new ...'); await db.execute('INSERT INTO production_items_new SELECT ...'); await db.execute('DROP TABLE production_items'); await db.execute('ALTER TABLE production_items_new RENAME TO production_items'); } }

WebDAV 云备份实现

为支持坚果云等 WebDAV 服务的备份功能，项目使用 Dio 实现 WebDAV 协议的文件操作， 包括身份验证、文件上传、下载、列表查询等核心功能。
